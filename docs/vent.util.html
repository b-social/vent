highest-arity-of
(highest-arity-of f)
Returns the maximum arity of:
  - anonymous functions like `#()` and `(fn [])`.
  - defined functions like `map` or `+`.
  - macros, by passing a var like `#'->`.

Returns `:variadic` if the function/macro is variadic.
